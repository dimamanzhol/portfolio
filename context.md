# Portfolio Website Design Context

## Overview

A two-page portfolio website:

1. **Homepage** - About me, services, recognition, contact
2. **Projects Page** - Showcase of all my work with detailed project views

## Page 1: Homepage Layout

**Header**

- Name centered at top (clean, minimalist)

**About Section**

- Short bio paragraph
- Role/title (design engineer, developer, etc.)
- Location

**Three-Column Grid**

- **Services** (left): List of what I offer (development, design, etc.)
- **Recognition** (middle): Awards, features, achievements
- **Contact** (right): Email and social media links

## Page 2: Projects Layout

**Project Display**

- Large, centered project cards
- Each project shows:
  - Visual preview (video)
  - Project title
  - Brief description
  - Project type/tags (web, mobile, etc.)
  - Link to live project
  - "View next project" navigation

## Design & Colors (Dev.style Inspired)

**Background**

- Dark charcoal (#1a1a2e to #121212)
- Subtle pixelated or mesh texture for tech vibe
- High contrast for readability

**Text Colors**

- White (#FFFFFF) for main text and headings
- Light gray (#a0a0a0) for secondary text and descriptions
- Darker gray (#666666) for muted elements

**Cards**

- Dark semi-transparent backgrounds
- Thin light gray borders (#333333)
- Minimal shadows
- Grid-based layout with consistent spacing

**Typography**

- **Headings**: Clean sans-serif (Inter or similar) - bold, white
- **Body text**: Regular sans-serif - light gray for descriptions
- **Code/Tech**: Monospace font (Fira Code or similar) for technical elements
- Generous line height (1.5-1.7) for readability

**Developer Aesthetic**

- CLI/command-line style elements (npx commands, code snippets)
- Dark theme matching code editors/terminals
- Technical credibility focus (minimal fluff, maximum functionality)
- Simple, functional UI elements (no unnecessary animations)
- Tech-forward, professional vibe

## Content Sections

1. **About Me** - Left side
   - Photo
   - Name
   - Short bio
   - Social links

2. **Projects** - Right side
   - Grid layout showing all projects
   - Each project as a card with visual preview
   - Project category labels at bottom of cards

3. **Work Experience**
   - Can be integrated into project cards or as separate section

## Spacing & Layout

- Generous white space between sections
- Consistent padding around cards
- Clean, uncluttered design
- Grid-based organization

## Overall Style

**Developer-focused minimalism** - Dark theme, high contrast, clean typography, and functional design. Similar to dev.style with:

- No unnecessary animations or fluff
- Technical credibility through simple, professional presentation
- Code/CLI elements to showcase developer skills
- Fast-loading, accessible design
- Content-first approach
